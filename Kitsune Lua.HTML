CHAPTER 1

Apa itu Lua?

Lua adalah bahasa pemrograman yang ringan, cepat, dan mudah dipelajari. Dibuat di Brazil tahun 1993, Lua sering dipakai di game (terutama Roblox), aplikasi, dan script otomatis.

🌙 Kenapa Belajar Lua?

Lua dipakai di Roblox Studio, game engine seperti Love2D, dan banyak aplikasi besar. Syntaxnya simpel dan mudah dimengerti bahkan untuk pemula.

✅ Lua adalah bahasa #1 yang dipakai di Roblox. Kalau kamu mau bikin game Roblox, kamu HARUS belajar Lua!

CHAPTER 2

Print & Output

Fungsi pertama yang harus kamu pelajari adalah print() — untuk menampilkan teks ke layar/konsol.

📢 Fungsi print()

print() adalah fungsi bawaan Lua untuk menampilkan output. Bisa menampilkan teks, angka, dan variabel.

LUA
print("Hello Dev!") print("Lua itu seru!") print(42) print(3.14) print(true)
OUTPUT
Hello Dev! Lua itu seru! 42 3.14 true

🔗 Menggabung Teks (Concatenation)

Di Lua, pakai .. untuk menggabungkan string.

LUA
local nama = "Kitsune" print("Halo, " .. nama .. "!") -- Multiple argumen dipisah koma (ada spasi otomatis) print("Nama:", nama, "| Channel: @KitsuneeIsMe")
OUTPUT
Halo, Kitsune! Nama: Kitsune | Channel: @KitsuneeIsMe

🖨️ io.write() vs print()

io.write() mencetak tanpa newline di akhir, berguna untuk output satu baris.

LUA
io.write("Hello ") io.write("Dev!\n") -- \n = newline manual print("Ini pakai print (ada enter otomatis)")
OUTPUT
Hello Dev! Ini pakai print (ada enter otomatis)
CHAPTER 3

Variabel & Tipe Data

Variabel adalah tempat menyimpan data. Di Lua ada beberapa tipe data dasar yang wajib kamu kenal.

📦 Membuat Variabel

Pakai kata kunci local untuk variabel lokal (disarankan). Tanpa local = variabel global.

LUA
-- Variabel lokal (DISARANKAN) local nama = "Kitsune" local umur = 17 local tinggi = 170.5 local aktif = true local kosong = nil -- nil = tidak ada nilai -- Variabel global (hindari kalau bisa) namaGlobal = "Ini global" print(nama, umur, aktif)
OUTPUT
Kitsune 17 true

LUA
print(type("Hello")) -- string print(type(42)) -- number print(type(true)) -- boolean print(type(nil)) -- nil print(type({})) -- table print(type(print)) -- function
OUTPUT

CHAPTER 4

Operator

Operator dipakai untuk melakukan operasi matematika, perbandingan, dan logika di Lua.

➕ Operator Matematika

LUA
local a = 10 local b = 3 print(a + b) -- 13 (tambah) print(a - b) -- 7 (kurang) print(a * b) -- 30 (kali) print(a / b) -- 3.3333... (bagi) print(a % b) -- 1 (sisa bagi / modulo) print(a ^ b) -- 1000 (pangkat) print(a // b) -- 3 (bagi bulat / floor div, Lua 5.3+)

⚖️ Operator Perbandingan

LUA
print(5 == 5) -- true (sama dengan) print(5 ~= 3) -- true (tidak sama) -- bukan != print(5 > 3) -- true (lebih besar) print(5 < 3) -- false (lebih kecil) print(5 >= 5) -- true (lebih besar atau sama) print(5 <= 4) -- false (lebih kecil atau sama)

🧠 Operator Logika

LUA
print(true and true) -- true (keduanya harus true) print(true and false) -- false print(true or false) -- true (salah satu true = true) print(false or false) -- false print(not true) -- false (kebalikan) print(not false) -- true
CHAPTER 5

String (Teks)

String adalah tipe data untuk menyimpan teks. Lua punya banyak fungsi bawaan untuk memanipulasi string.

🔧 Fungsi-Fungsi String

LUA
local s = "Hello Lua Dev!" print(#s) -- 14 (panjang string) print(string.len(s)) -- 14 print(string.upper(s)) -- HELLO LUA DEV! print(string.lower(s)) -- hello lua dev! print(string.sub(s, 1, 5)) -- Hello (potong karakter 1-5) print(string.rep("Ha", 3)) -- HaHaHa (ulangi 3x) print(string.reverse(s)) -- !veD auL olleH print(string.find(s, "Lua")) -- 7 9 (posisi ditemukan) print(string.gsub(s, "Dev", "World")) -- Hello Lua World! -- Format string (seperti printf) print(string.format("Nama: %s, Umur: %d", "Kitsune", 17))
OUTPUT
Nama: Kitsune, Umur: 17
CHAPTER 6

If / Else

Percabangan memungkinkan program mengambil keputusan berdasarkan kondisi tertentu.

❓ Struktur If / Elseif / Else

LUA
local nilai = 85 if nilai >= 90 then print("A - Sangat Bagus!") elseif nilai >= 80 then print("B - Bagus!") elseif nilai >= 70 then print("C - Cukup") else print("D - Perlu Belajar Lagi") end -- Kondisi dengan and / or local hp = 50 local shield = false if hp > 0 and not shield then print("Player masih hidup, tidak ada shield") end
OUTPUT
B - Bagus! Player masih hidup, tidak ada shield

💡 Shortcut: and / or sebagai Ternary

LUA
-- Di bahasa lain: x = kondisi ? a : b -- Di Lua: local hidup = true local status = hidup and "Hidup" or "Mati" print(status) -- Hidup
CHAPTER 7

Loop / Perulangan

Loop membuat kode dijalankan berulang-ulang. Lua punya 3 jenis loop utama: for, while, dan repeat.

🔢 For Loop (Numerik)

LUA
-- for mulai, selesai, langkah do for i = 1, 5 do print("Step ke-" .. i) end -- Hitung mundur (langkah -1) for i = 5, 1, -1 do print(i) end -- Lompat 2-2 (step 2) for i = 0, 10, 2 do io.write(i .. " ") end
OUTPUT
Step ke-1 ... Step ke-5 5 4 3 2 1 0 2 4 6 8 10

🔄 While & Repeat

LUA
-- While: cek kondisi SEBELUM jalankan local n = 1 while n <= 5 do print("while: " .. n) n = n + 1 end -- Repeat: jalankan DULU baru cek kondisi local x = 0 repeat x = x + 1 print("x = " .. x) until x >= 3 -- Break: hentikan loop lebih awal for i = 1, 100 do if i == 5 then break end print(i) end
CHAPTER 8

Fungsi / Functions

Fungsi adalah blok kode yang bisa dipanggil berulang kali. Sangat penting untuk membuat kode yang terorganisir dan efisien.

⚙️ Membuat dan Memanggil Fungsi

LUA
-- Fungsi sederhana local function sapa() print("Halo dari fungsi!") end sapa() -- panggil fungsinya -- Fungsi dengan parameter local function sapaNama(nama) print("Halo, " .. nama .. "!") end sapaNama("Kitsune") -- Fungsi dengan return (nilai balik) local function tambah(a, b) return a + b end local hasil = tambah(5, 3) print("Hasil: " .. hasil) -- Fungsi return multiple values! local function minmax(a, b) return math.min(a, b), math.max(a, b) end local kecil, besar = minmax(10, 3) print(kecil, besar) -- 3 10

🎯 Variadic Function (...)

Fungsi yang bisa menerima jumlah argumen tak terbatas.

LUA
local function jumlahSemua(...) local args = {...} local total = 0 for _, v in ipairs(args) do total = total + v end return total end print(jumlahSemua(1, 2, 3, 4, 5)) -- 15
CHAPTER 9

Tables

Table adalah struktur data paling penting di Lua. Bisa berfungsi sebagai array, dictionary (kamus), set, dan bahkan OOP class!

📋 Table sebagai Array

LUA
-- Array di Lua dimulai dari index 1 (bukan 0!) local buah = {"apel", "mangga", "jeruk"} print(buah[1]) -- apel print(buah[2]) -- mangga print(#buah) -- 3 (panjang array) -- Tambah elemen table.insert(buah, "durian") -- Hapus elemen table.remove(buah, 1) -- hapus index 1 -- Loop array pakai ipairs for i, v in ipairs(buah) do print(i, v) end

🗂️ Table sebagai Dictionary

LUA
local player = { nama = "Kitsune", hp = 100, level = 25, aktif = true } print(player.nama) -- Kitsune print(player["hp"]) -- 100 (cara lain) -- Ubah nilai player.hp = 80 -- Loop dictionary pakai pairs for key, val in pairs(player) do print(key .. " = " .. tostring(val)) end
CHAPTER 10

OOP & Class di Lua

Lua tidak punya class bawaan, tapi kita bisa membuat OOP (Object-Oriented Programming) menggunakan table dan metatables.

🏗️ Membuat Class dengan Table

LUA
-- Buat "class" Player local Player = {} Player.__index = Player -- Constructor function Player.new(nama, hp) local self = setmetatable({}, Player) self.nama = nama self.hp = hp return self end -- Method function Player:info() print("Nama: " .. self.nama .. " | HP: " .. self.hp) end function Player:takeDamage(damage) self.hp = self.hp - damage print(self.nama .. " kena damage " .. damage) end -- Buat instance local p1 = Player.new("Kitsune", 100) p1:info() p1:takeDamage(30) p1:info()
OUTPUT
Nama: Kitsune | HP: 100 Kitsune kena damage 30 Nama: Kitsune | HP: 70

⏱️ Wait, Delay, dan Spawn

ROBLOX LUA
-- Pause eksekusi selama N detik task.wait(2) -- tunggu 2 detik (pakai task.wait, bukan wait()) -- Jalankan setelah delay tanpa blocking task.delay(3, function() print("Ini dijalankan 3 detik kemudian") end) -- Jalankan di thread baru (tidak blocking) task.spawn(function() while true do print("Loop terus di background!") task.wait(1) end end) print("Kode ini tetap jalan!")
💡 Selalu pakai task.wait() bukan wait() di Roblox modern — lebih efisien dan akurat!
ROBLOX GUI

Frame & GUI di Roblox

GUI (Graphical User Interface) di Roblox dibuat pakai ScreenGui, Frame, TextLabel, TextButton, dan elemen lainnya via kode Lua.

🖼️ Membuat ScreenGui & Frame dari Script

ROBLOX LOCAL SCRIPT
local Players = game:GetService("Players") local LocalPlayer = Players.LocalPlayer local PlayerGui = LocalPlayer:WaitForChild("PlayerGui") -- 1. Buat ScreenGui local screenGui = Instance.new("ScreenGui") screenGui.Name = "MyGui" screenGui.ResetOnSpawn = false screenGui.Parent = PlayerGui -- 2. Buat Frame (container utama) local frame = Instance.new("Frame") frame.Size = UDim2.new(0, 300, 0, 200) frame.Position = UDim2.new(0.5, -150, 0.5, -100) frame.BackgroundColor3 = Color3.fromRGB(30, 30, 40) frame.BorderSizePixel = 0 frame.Parent = screenGui -- 3. Tambah sudut melengkung local corner = Instance.new("UICorner") corner.CornerRadius = UDim.new(0, 12) corner.Parent = frame -- 4. Buat TextLabel (teks biasa) local label = Instance.new("TextLabel") label.Size = UDim2.new(1, 0, 0, 50) label.Position = UDim2.new(0, 0, 0, 0) label.Text = "🌙 LuaLearn GUI!" label.TextColor3 = Color3.fromRGB(255, 255, 255) label.TextSize = 18 label.BackgroundTransparency = 1 label.Font = Enum.Font.GothamBold label.Parent = frame -- 5. Buat TextButton (tombol) local btn = Instance.new("TextButton") btn.Size = UDim2.new(0.8, 0, 0, 40) btn.Position = UDim2.new(0.1, 0, 0.6, 0) btn.Text = "Klik Aku!" btn.BackgroundColor3 = Color3.fromRGB(124, 106, 247) btn.TextColor3 = Color3.fromRGB(255, 255, 255) btn.Font = Enum.Font.GothamBold btn.TextSize = 14 btn.BorderSizePixel = 0 btn.Parent = frame -- 6. Event saat tombol diklik btn.MouseButton1Click:Connect(function() label.Text = "Tombol diklik! ✅" print("Tombol diklik!") end)

📐 Memahami UDim2

UDim2 adalah cara Roblox mengatur ukuran dan posisi. Formatnya: UDim2.new(scaleX, offsetX, scaleY, offsetY)

UDim2Artinya
UDim2.new(1, 0, 1, 0)Penuh layar (100% x 100%)
UDim2.new(0.5, 0, 0.5, 0)Tengah layar (50% x 50%)
UDim2.new(0, 200, 0, 50)200px lebar, 50px tinggi (pixel murni)
UDim2.new(0.5, -100, 0, 0)50% layar dikurangi 100px (untuk centering)

SCRIPT POPULER

Auto Farm

Auto Farm adalah script yang membuat karakter otomatis mengumpulkan item atau mengalahkan musuh secara berulang tanpa input manual.

⚠️ Script ini hanya untuk pembelajaran. Penggunaan exploit di game orang lain bisa menyebabkan ban. Gunakan di private server atau game buatan sendiri!

🌾 Template Auto Farm (Teleport ke NPC)

ROBLOX LUA
local Players = game:GetService("Players") local lp = Players.LocalPlayer local farming = false local function getChar() return lp.Character end local function teleportTo(position) local char = getChar() if char then local root = char:FindFirstChild("HumanoidRootPart") if root then root.CFrame = CFrame.new(position) end end end local function autoFarm() while farming do -- Cari semua NPC/Mob di workspace for _, obj in pairs(workspace:GetDescendants()) do if not farming then break end -- Ganti "Zombie" dengan nama NPC di game kamu if obj.Name == "Zombie" and obj:IsA("Model") then local hum = obj:FindFirstChild("Humanoid") if hum and hum.Health > 0 then local root = obj:FindFirstChild("HumanoidRootPart") if root then teleportTo(root.Position) task.wait(0.1) end end end end task.wait(0.5) end end -- Toggle dengan tombol F local UIS = game:GetService("UserInputService") UIS.InputBegan:Connect(function(inp, gp) if gp then return end if inp.KeyCode == Enum.KeyCode.F then farming = not farming print("Auto Farm: " .. tostring(farming)) if farming then task.spawn(autoFarm) end end end)

ESP / Highlight

ESP (Extra Sensory Perception) membuat player atau objek terlihat melalui dinding menggunakan Highlight atau BillboardGui.

👁️ ESP menggunakan Highlight

ROBLOX LUA
local Players = game:GetService("Players") local LocalPlayer = Players.LocalPlayer local function addESP(player) if player == LocalPlayer then return end player.CharacterAdded:Connect(function(char) task.wait(0.5) local hl = Instance.new("Highlight") hl.FillColor = Color3.fromRGB(255, 0, 0) hl.OutlineColor = Color3.fromRGB(255, 255, 255) hl.FillTransparency = 0.5 hl.OutlineTransparency = 0 hl.DepthMode = Enum.HighlightDepthMode.AlwaysOnTop hl.Adornee = char hl.Parent = char end) -- Kalau sudah punya karakter if player.Character then local hl = Instance.new("Highlight") hl.FillColor = Color3.fromRGB(255, 0, 0) hl.OutlineColor = Color3.fromRGB(255, 255, 255) hl.FillTransparency = 0.5 hl.DepthMode = Enum.HighlightDepthMode.AlwaysOnTop hl.Adornee = player.Character hl.Parent = player.Character end end -- Pasang ke semua player yang sudah ada for _, p in ipairs(Players:GetPlayers()) do addESP(p) end -- Pasang ke player yang join setelahnya Players.PlayerAdded:Connect(addESP)

Speed & Jump Hack

Script untuk mengubah kecepatan lari dan tinggi lompatan karakter di Roblox.

💨 Speed & Jump Script dengan Toggle GUI

ROBLOX LUA
local Players = game:GetService("Players") local lp = Players.LocalPlayer -- Konfigurasi local speedValue = 50 -- Default 16 local jumpValue = 80 -- Default 50 local speedOn = false local jumpOn = false local function getHumanoid() local char = lp.Character return char and char:FindFirstChild("Humanoid") end local function applySpeed(on) local hum = getHumanoid() if hum then hum.WalkSpeed = on and speedValue or 16 end end local function applyJump(on) local hum = getHumanoid() if hum then hum.JumpPower = on and jumpValue or 50 end end -- Terapkan ulang saat spawn lp.CharacterAdded:Connect(function() task.wait(0.5) applySpeed(speedOn) applyJump(jumpOn) end) -- Toggle dengan tombol local UIS = game:GetService("UserInputService") UIS.InputBegan:Connect(function(inp, gp) if gp then return end if inp.KeyCode == Enum.KeyCode.Q then speedOn = not speedOn applySpeed(speedOn) print("Speed: " .. tostring(speedOn)) elseif inp.KeyCode == Enum.KeyCode.J then jumpOn = not jumpOn applyJump(jumpOn) print("Jump: " .. tostring(jumpOn)) end end)
🧠 SCRIPT POPULER

Auto Find Brainrots

Script untuk otomatis menemukan dan berinteraksi dengan Brainrot entities di game Roblox. Konsep: scan workspace, filter berdasarkan nama, teleport dan interaksi.

🧠 Script ini menunjukkan cara scan workspace, filter model berdasarkan nama, dan auto-interact — konsep yang bisa dipakai di banyak game Roblox!

🔍 Auto Find & Interact Script

ROBLOX LUA
local Players = game:GetService("Players") local lp = Players.LocalPlayer local running = false -- Daftar nama Brainrot yang dicari (sesuaikan dengan game) local BRAINROT_NAMES = { "Tralalero", "Bombardiro", "Capybara", "Tung Tung", "Ballerina", "Crocodilo" } -- Cek apakah nama ada di list local function isBrainrot(name) for _, brName in ipairs(BRAINROT_NAMES) do if name:lower():find(brName:lower()) then return true end end return false end -- Teleport karakter ke posisi tertentu local function tpTo(pos) local char = lp.Character if char then local root = char:FindFirstChild("HumanoidRootPart") if root then root.CFrame = CFrame.new(pos + Vector3.new(0, 3, 0)) end end end -- Cari semua brainrot di workspace local function findBrainrots() local found = {} for _, obj in pairs(workspace:GetDescendants()) do if obj:IsA("Model") and isBrainrot(obj.Name) then local root = obj:FindFirstChild("HumanoidRootPart") or obj:FindFirstChild("PrimaryPart") if root then table.insert(found, {model = obj, pos = root.Position}) end end end return found end -- Main loop local function autoFind() while running do local list = findBrainrots() print("Ditemukan: " .. #list .. " brainrot") for _, data in ipairs(list) do if not running then break end print("Menuju: " .. data.model.Name) tpTo(data.pos) task.wait(1) -- waktu interaksi end task.wait(2) end end -- Toggle dengan tombol B local UIS = game:GetService("UserInputService") UIS.InputBegan:Connect(function(inp, gp) if gp then return end if inp.KeyCode == Enum.KeyCode.B then running = not running print("Auto Brainrot: " .. tostring(running)) if running then task.spawn(autoFind) end end end)
ROBLOX ADVANCED

Remote Events

RemoteEvent dipakai untuk komunikasi antara Client (LocalScript) dan Server (Script). Sangat penting untuk membuat fitur multiplayer.

📡 Cara Kerja RemoteEvent

SERVER SCRIPT
-- Di Script (Server) — buat RemoteEvent local RS = game:GetService("ReplicatedStorage") local event = Instance.new("RemoteEvent") event.Name = "MyEvent" event.Parent = RS -- Terima event dari client event.OnServerEvent:Connect(function(player, data) print(player.Name .. " kirim: " .. tostring(data)) -- Kirim balik ke semua client event:FireAllClients("Server menerima: " .. tostring(data)) end)
LOCAL SCRIPT (CLIENT)
-- Di LocalScript (Client) — gunakan RemoteEvent local RS = game:GetService("ReplicatedStorage") local event = RS:WaitForChild("MyEvent") -- Kirim ke server event:FireServer("Halo Server!") -- Terima dari server event.OnClientEvent:Connect(function(msg) print("Dari server: " .. msg) end)
💡 Selalu validasi data di server! Jangan percaya data yang dikirim client begitu saja — karena bisa dimanipulasi oleh exploiter.
